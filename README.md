
# EchoMessenger
Ｃ＃코딩　에코　메신저

📌1. 개요 (Overview) C# WinForms를 이용하여 단계별로 기능을 확장하며 제작한 메신저 스타일 프로그램입니다. 기초적인 UI 배치부터 복잡한 예외 처리 및 데이터 관리까지의 과정을 기록하였습니다.

📅 2. 단계별 과제 수행 내용

📂 과제 1: 기본 UI 및 데이터 연동 목표: 기본적인 컨트롤 배치와 데이터 흐름 이해

주요 기능: * 
Label(표시), TextBox(입력), Button(전송), ListBox(대화창)를적절히배치합니다.

전송버튼클릭시TextBox의텍스트를ListBox의항목(Items)으로추가합니다.

전송 후 입력창 자동 비우기(Clear)

📂 과제 2: 사용자 편의 기능 및 입력 방어 목표: 실질적인 사용성을 높이고 잘못된 입력을 차단

주요 기능:

엔터키 전송: 마우스 클릭 없이 Enter 키만으로 전송 가능 

입력 방어: 공백이나 빈 문자열 전송 차단 

자동 포커스: 전송 후 커서를 입력창으로 자동 이동

📂 과제 3: 문자열 처리 및 시스템 연동 목표: 데이터 가공 및 실시간 정보 표시

주요 기능:

타임스탬프: 메시지마다 현재 시간을 [HH:mm:ss] 형식으로 부착

메시지 카운팅: 현재 총 대화 개수를 하단 레이블에 실시간 표시

문자열 정제: Trim() 함수로 불필요한 앞뒤 공백 제거
